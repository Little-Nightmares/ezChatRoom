# 局域网即时通讯 - 项目结构说明

## 项目概述

这是一个基于 **Qt + C++** 构建的局域网即时通讯系统，采用 **客户端-服务器（C/S）架构**，支持账号登录、文字聊天、好友管理、消息加密和离线留言功能。

### 技术栈

| 技术    | 版本       | 用途                  |
| ------- | ---------- | --------------------- |
| Qt/QML  | 6.11.0     | GUI 框架、网络通信    |
| C++     | C++17      | 核心业务逻辑          |
| OpenSSL | 3.x        | 消息加密（AES / RSA） |
| SQLite  | 3.x        | 本地聊天记录存储      |
| JSON    | -          | 数据交换格式          |

详细目录结构请参见项目根目录下的 README.md 及各源文件旁的 .md 说明文件。
